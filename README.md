# JavaScript To-Do List Application

## Overview

This project is a simple and functional To-Do List web application developed using HTML, CSS, and JavaScript. The application allows users to manage daily tasks by adding new tasks, deleting tasks, and marking tasks as completed.

The project was created to practice JavaScript fundamentals, DOM manipulation, event handling, and browser Local Storage functionality.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Store tasks using browser Local Storage
- Maintain tasks after refreshing the page
- Simple and user-friendly interface
- Responsive layout for different screen sizes

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Browser Local Storage

## How It Works

1. Users can enter a task using the input field.
2. New tasks are added to the task list dynamically.
3. Users can mark tasks as completed.
4. Users can remove tasks when they are no longer needed.
5. Tasks are saved in Local Storage so they remain available after refreshing or reopening the browser.

## Project Purpose

The purpose of this project was to develop practical frontend development skills by creating an interactive web application using JavaScript. It helped improve understanding of DOM manipulation, JavaScript functions, and handling user interactions.
